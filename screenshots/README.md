# Secure User Authentication System

This project implements a secure user authentication system using Flask.

## Features
- User Registration & Login
- Password Hashing
- Session Management
- Protected Routes
- Admin Dashboard
- About Page

## Tech Stack
- Python
- Flask
- SQLAlchemy
- Bootstrap
- SQLite
